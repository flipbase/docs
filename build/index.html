Introduction

Flipbase developer documentation

Flipbase enables you to add video recording and playback capabilities to your website, web application or recruitment-system, with 1 line of Javascript code. Flipbase exists of a recorder.js Javascript library, a player.js Javascript library, Flipbase Platform and an API.

Scope of the documentation

The documentation should you guide through some basic concepts of our service and how you can use the different parts of the Flipbase service. The documentation assumes knowledge of programming and networking.
