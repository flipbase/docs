Introduction

Flipbase documentation

Flipbase enables you to add video recording and playback capabilities to your website, web application or recruitment-system, with 1 line of Javascript code. Flipbase exists of a recorder.js Javascript library, a player.js Javascript library and an API for partners.

In order to use Flipbase you will need a Flipbase account. We will create a Flipbase account for you and provide you with your unique Recorder and Player Id’s. When you want to use the API you need an client_id and a client_secret, provided by Flipbase.
